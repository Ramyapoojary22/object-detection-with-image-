# Object Detection with Images
## Description
This project detects objects in images using the YOLOv8 model. Users can upload an image through a Flask web application, and the system detects objects and displays the result with bounding boxes.
## Technologies Used
- Python
- Flask
- YOLOv8
- OpenCV
- SQLite
- HTML
- CSS
## Features
- User Login & Registration
- Image Upload
- Object Detection
- Displays Detected Image with Bounding Boxes
## How to Run
1. Install the required libraries.
2. Run:
   python app.py
3. Open the application in your browser.
